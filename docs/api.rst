Pywinauto recorder API
======================

Setup
-----
If you want to replay the actions of a script generated by "pywinauto_recorder.exe_" in your Pyhton interpreter you must install the pywinauto_recorder_ package:

.. _pywinauto_recorder: https://pypi.org/project/pywinauto-recorder/
.. _pywinauto_recorder.exe: https://raw.githubusercontent.com/beuaaa/pywinauto_recorder/master/bin/Pywinauto_recorder_installer.exe

.. code-block:: python

    pip install -U pywinauto_recorder
